# 📖 Guia do Usuário - HelpDesk Pro

## 🎯 Começando

### Navegação Principal

O sistema possui duas seções principais acessíveis pelo menu superior:

1. **Dashboard** 📊 - Visão geral e estatísticas
2. **Chamados** 🎫 - Lista e gerenciamento de chamados

---

## 🆕 Como Abrir um Novo Chamado

1. Clique no botão **"Novo Chamado"** (canto superior direito)
2. Preencha o formulário:
   - **Título**: Resumo do problema (ex: "Impressora não funciona")
   - **Descrição**: Detalhes completos do problema
   - **Categoria**: Selecione o tipo (Hardware, Software, Rede, etc.)
   - **Prioridade**: 
     - 🟢 **Baixa**: Pode aguardar
     - 🟡 **Média**: Problema comum
     - 🟠 **Alta**: Precisa de atenção rápida
     - 🔴 **Urgente**: Crítico, bloqueando trabalho
   - **Seu Nome**: Nome completo
   - **Seu Email**: Email para contato
3. Clique em **"Criar Chamado"**

### ✅ Dicas para um Bom Chamado

- **Seja específico no título**: "Computador lento" é melhor que "Problema"
- **Detalhe o problema**: O que aconteceu? Quando começou? Que mensagens aparecem?
- **Informe o que já tentou**: "Já reiniciei o computador" ajuda o técnico
- **Escolha a categoria correta**: Facilita o direcionamento
- **Defina a prioridade realista**: Urgente só se realmente bloquear seu trabalho

---

## 🔍 Como Usar os Filtros

1. Na seção **Chamados**, clique no botão **"Filtros"**
2. Selecione os filtros desejados:
   - **Buscar**: Digite palavras-chave
   - **Status**: Filtre por Aberto, Em Andamento, etc.
   - **Prioridade**: Veja apenas urgentes ou alta prioridade
   - **Categoria**: Filtre por tipo de problema
   - **Solicitante**: Encontre seus chamados (digite seu nome)
   - **Data**: Filtre por período
3. Clique em **"Aplicar Filtros"**
4. Para remover filtros: Clique em **"Limpar Filtros"**

### 🎯 Filtros Rápidos Úteis

- **Meus chamados abertos**: Filtrar por seu nome + Status "Aberto"
- **Urgentes**: Filtrar apenas Prioridade "Urgente"
- **Esta semana**: Usar filtro de data dos últimos 7 dias

---

## 👀 Como Acompanhar seu Chamado

1. Na seção **Chamados**, encontre seu ticket na lista
2. Clique no card do chamado para abrir os detalhes
3. Você verá:
   - **Status atual**: Onde está o atendimento
   - **Técnico responsável**: Quem está cuidando
   - **Comentários**: Atualizações do técnico
   - **Histórico**: Todas as mudanças

### 📊 Entendendo os Status

- 🟡 **Aberto**: Chamado criado, aguardando atendimento
- 🔵 **Em Andamento**: Técnico está trabalhando na solução
- 🟣 **Aguardando**: Dependendo de algo externo
- 🟢 **Resolvido**: Problema foi solucionado
- ⚫ **Fechado**: Chamado encerrado

---

## 💬 Como Adicionar Comentários

Ao visualizar os detalhes de um chamado:

1. Role até a seção **"Comentários"**
2. Digite sua mensagem no campo de texto
3. Clique em **"Adicionar Comentário"**

**Use comentários para:**
- Adicionar informações que esqueceu
- Responder perguntas do técnico
- Confirmar que o problema foi resolvido
- Informar mudanças na situação

## 🎓 Perguntas Frequentes

### Como sei se meu chamado foi visto?
Verifique se um técnico foi atribuído ou se há comentários.

### Posso editar um chamado após criar?
Não diretamente, mas você pode adicionar comentários com novas informações.

### Como sei quando está resolvido?
O status mudará para "Resolvido" e você receberá um comentário confirmando.

### Posso ver chamados de outros usuários?
Sim, todos os chamados são visíveis. Use filtros para ver apenas os seus.

### Como priorizar melhor?
- **Urgente**: Sistema fora do ar, bloqueio total de trabalho
- **Alta**: Problema sério mas há alternativa temporária
- **Média**: Problema comum, incomoda mas dá pra trabalhar
- **Baixa**: Melhoria ou problema menor

### O que fazer se a urgência mudar?
Adicione um comentário explicando a mudança de urgência. O técnico pode ajustar.

### Quanto tempo leva para ser atendido?
Depende da prioridade e disponibilidade. Veja o "Tempo Médio de Resolução" no Dashboard.

## 🆘 Problemas com o Sistema

Se o próprio sistema de chamados apresentar problemas:

1. Tente recarregar a página (F5)
2. Limpe o cache do navegador
3. Tente em outro navegador
4. Contate o administrador do sistema

## 💡 Dicas Profissionais

### Para Solicitar Atendimento Rápido
1. ✅ Título claro e objetivo
2. ✅ Descrição completa com detalhes
3. ✅ Categoria correta
4. ✅ Prioridade adequada
5. ✅ Responda prontamente aos comentários do técnico

### Para Organizar Seus Chamados
- Use filtros para encontrar rapidamente
- Verifique o Dashboard para visão geral
- Adicione comentários com atualizações
- Confirme quando o problema for resolvido

### Boas Práticas
- 📝 **Um problema por chamado**: Não misture múltiplas questões
- 🖼️ **Descreva visualmente**: "Botão vermelho no canto superior direito"
- ⏰ **Informe horários**: "Acontece sempre às 14h"
- 📊 **Mencione impacto**: "Afetando 5 pessoas do setor"
- ✉️ **Email correto**: Para receber notificações (quando disponível)
