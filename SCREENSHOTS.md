## 🎨 Paleta de Cores

### Status
- 🟡 **Aberto**: Yellow (bg-yellow-100, text-yellow-800)
- 🔵 **Em Andamento**: Blue (bg-blue-100, text-blue-800)
- 🟣 **Aguardando**: Purple (bg-purple-100, text-purple-800)
- 🟢 **Resolvido**: Green (bg-green-100, text-green-800)
- ⚫ **Fechado**: Gray (bg-gray-100, text-gray-800)

### Prioridade
- 🟢 **Baixa**: Green (bg-green-500)
- 🟡 **Média**: Yellow (bg-yellow-500)
- 🟠 **Alta**: Orange (bg-orange-500)
- 🔴 **Urgente**: Red (bg-red-500)

### Ícones Principais
- 🏠 Home (Dashboard)
- 🎫 Ticket (Chamados)
- ➕ Plus (Novo)
- 🔍 Search (Busca)
- 🔔 Bell (Notificações)
- 👤 User (Usuário)
- 🕐 Clock (Tempo)
- 💬 Chat (Comentários)
- 🏷️ Tag (Categoria)
- ✅ Check (Concluído)
- ⚠️  Warning (Alerta)

## 🖱️ Interações

### Hover Effects
```
Normal:     ┌──────────┐
            │  Card    │
            └──────────┘

Hover:      ┌──────────┐  ← Shadow aumenta
            │  Card    │  ← Borda muda de cor
            └──────────┘  ← Cursor pointer
```

### Click States
```
Button Normal:    [  Criar Chamado  ]

Button Hover:     [  Criar Chamado  ]  ← Cor mais escura

Button Active:    [  Criar Chamado  ]  ← Ligeiramente menor

Button Disabled:  [  Criando...     ]  ← Opacidade reduzida
```
